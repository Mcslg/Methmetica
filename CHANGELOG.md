# 更新日誌 (CHANGELOG)

所有對 **Methmetica** 的重大變更都將記錄在此。

## [0.1.0] - 2026-03-14

### ✨ 新功能 (Added)
- **視覺化節點架構**: 建立核心 React Flow 畫布，支援節點與連線。
- **動態 Handle 系統**: 實作可在節點邊緣任何位置新增/刪除連接點的功能。
- **徑向輪盤選單 (Radial Menu)**: 優化 UX，透過環形介面快速選擇點位類型。
- **自動感應旋轉**: Handle 圖示會根據放置位置自動旋轉（Input 指向內部，Output 指向外部）。
- **數據/工具節點區分**: 
  - `NumberNode`: 支援手動輸入或接收上游數據。
  - `AddNode`: 工具類節點，具備 `EXEC` 手動觸發執行按鈕。
- **右鍵選單增強**: 
  - 空白處右鍵：新增節點。
  - 邊緣右鍵：新增 Handle。
  - 節點右鍵：刪除節點。
- **專案文化**: 導入 **Vibe-coding** 精神與精美 README 文件。

### 🎨 視覺與 UI (Aesthetics)
- **玻璃美學 (Glassmorphism)**: 節點具備半透明毛玻璃質感。
- **動態指標**: 邊緣感應區顯示十字準心 (Crosshair)，提示可互動區域。
- **顏色編碼**: 使用不同邊框樣式區分 數據型 vs 工具型 節點。

### ⚙️ 底層更新 (Infrastructure)
- **GitHub 初始化**: 專案成功推送到 GitHub 遠端倉庫。
- **數學引擎準備**: 安裝 MathLive, Nerdamer, KaTeX 為符號運算版本做準備。
