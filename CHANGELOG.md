# 更新日誌 (CHANGELOG)

所有對 **Methmetica** 的重大變更都將記錄在此。

## [0.6.1] - 2026-03-21

### ✨ 交互與 UI 優化 (Interactive & UI Polish)
- **Math Pill 文字選取與複製**:
    - 徹底優化了 Math Pill 的 DOM 結構，現在可以像普通文字一樣**滑鼠框選**。
    - 支援選取時的**視覺反饋**（藍色/綠色選取背景），並修復了根號、分式無法被選取到的問題。
    - 支援 **onCopy 自定義複製**，複製出來的內容保證是純 LaTeX 文字。
- **游標跟隨編輯面板 (Caret-Following Popup)**:
    - 重寫了數學藥丸的編輯介面，彈窗現在會自動出現在**文字閃爍游標 (Caret)** 旁，大幅縮減視線偏移。
    - **UI 迷你化**：重組面板佈局，將「變數名稱」與「控制按鈕」併至底端，騰出更多空間給公式輸入。
- **智慧型垂直聯立方程建議**:
    - 自動偵測公式首字（相容 `{`, `\left\lbrace` 等符號），若符合聯立特徵自動轉化為垂直的 `cases` 樣式。
- **藥丸拆解功能 (Pill Explode)**:
    - 實作按住 **`Alt + 左鍵`** 快速拆解：可將計算結果（如 `[x=3, y=5]`）瞬間分裂成兩個獨立的變數定義藥丸（`x:3`, `y:5`）。

### 🐛 修復 (Fixed)
- **輸入穩定性**: 修復了在特定情況下按下 Save 無法正確插入新藥丸的 Tiptap 選取區錯誤。
- **點擊衝突解決**: 透過計算滑鼠位移距離，解決了點擊藥丸編輯與框選文字動作之間的偵測衝突。

## [0.6.0] - 2026-03-20

### ✨ 新功能 (Added)
- **GraphNode 繪圖引擎升級**:
    - **雙層求值架構**: 優先使用 CortexJS `compile()` 進行高效運算，並自動回退到具備「隱式乘法」支援的 JS `eval` 引擎，解決 `2x` 無法畫圖的問題。
    ~~- **Shift 鍵精確符號座標**: 實作按住 `Shift` 顯示關鍵點（零點、極值）的解析解（如 `(√2, 0)`、`(π, 0)`），不再只是小數。~~還是沒有用
    - **磁性吸附自動傳參**: 節點靠近時會自動擷取相鄰節點的公式（TextNode 的藥丸或 CalculateNode 的結果），並自動切換至 `EXT` 外部公式模式開始繪圖。
- **TextNode 數據擷取與運算強化**:
    - **改用tiptap引擎**: 支援無縫編輯，接口錯開，藥丸可以命名。
    - **局部變數作用域 (Local Scope)**: 現在藥丸 (Math Pill) 運算時會自動感應同一個 TextNode 內的其他具名變數，支援如「a=2, b=3, a+b=5」的連鎖計算。
    - **CortexJS 引擎切換**: 運算後台由 Nerdamer 全面遷移至 CortexJS，提升 LaTeX 解析精度與運算穩定性。
    - **磁性吸附支援**: 支援將 TextNode 內部的所有「數學藥丸」值合併後透過磁性吸附一次性傳遞給 GraphNode。
- **全域計算服務 (CalculationService) 升級**:
    - **微積分支援**: 微分 (Derivatives) 與積分 (Integrals) 現在使用 CortexJS 原生盒裝表達式 (`ce.box`) 運算，精度更高且支援更多特殊函數。
    - **節點同步**: 將 GraphNode 納入計算流，確保數據在不同節點間的傳遞始終與畫布狀態同步。

### 🎨 視覺與 UI (Aesthetics)
- **磁性吸附視覺反饋**: 節點吸附時，受影響的邊緣會顯示發光藍色邊框（`edge-touch-*`），與連接線視覺風格統一。
- **GraphNode 外部模式標示**: 當接收到外部公式時，頂部顯示具備發光效果的 `● EXT` 標記。

### 🐛 修復 (Fixed)
- **公式解析容錯**: 修復了帶等號公式（如 `y = x^2`）解析失敗的問題，現在會自動提取右側表達式。
- **變數污染修復**: 修正了底層 Bug 導致 `x` 被鎖定為常數而無法畫出斜線的問題。
- **清除邏輯**: 修復了分離吸附節點後公式輸入 (formulaInput) 未正確重設的問題。

## [0.5.0] - 2026-03-18

### ✨ 新功能 (Added)
- **GraphNode 智慧標記系統**:
    - **自動偵測關鍵點**: 使用數值掃描法（Sign-Change Detection）在 x ∈ [-20, 20] 範圍內自動偵測並標記：
        - **零點 (Roots)**: 曲線與 X 軸的交點
        - **極值點 (Extrema)**: 函數的波峰與波谷
        - **交點 (Intersections)**: 多條曲線彼此的交叉點
    - ~~**Shift 鍵切換符號**: 按住 `Shift` 可從小數坐標切換為精確符號算式（如 `(√2, 0)`）~~ 尚未實施
    - **跨類型去重**: 當零點與交點重合時，只顯示一個標記點，避免重疊
- **SliderNode（滑桿節點）**: 新增可供使用者手動調整數值的互動式滑桿，支援自訂最小值、最大值與步進，並透過 `val` 接環輸出

### 🎨 視覺與 UI (Aesthetics)
- **右鍵選單雙排顯示**: 節點建立選單現在以雙排網格顯示，大幅降低垂直高度
- **選單溢出防護**: 選單現在會自動偵測畫面邊緣，貼近螢幕邊時自動往內偏移
- **Shift 顯示節點說明**: 開啟右鍵選單時按住 `Shift` 可展開所有節點的詳細說明文字；搜尋時也會自動展開
- **節點精簡化**:
    - **CalculateNode**: 移除多餘的 `Formula:` 標籤並縮減內距，更加緊湊
    - **Trigger（ButtonNode）**: 移除 `RUN` 標籤，預設尺寸縮至 120×46，外觀更精簡
    - **Calculus（Differentiate/Integrate）**: 移除空白內容區、縮小控制列，預設尺寸縮至 160×75

### 🐛 修復 (Fixed)
- **GraphNode 縮放跳動**: 修正 Y 軸計算錯誤，現在滾輪縮放以滑鼠數學座標為中心，完全不再跳動
- **GraphNode 調整大小**: 修復調整節點大小時兩側同時移動的問題，與修復拖拉點被裁切在節點內部的問題
- **GraphNode 內部滾輪失效**: 修復新增內層 wrapper 後 `nowheel` 遮蔽事件導致圖形縮放失效的問題
- **關鍵點掃描精度**: 修復步長剛好落在整數座標（如 x=0、x=1）時，交點/零點被遺漏的問題

## [0.4.0] - 2026-03-16

### ✨ 新功能 (Added)
- **ForEachNode（迭代節點）**: 對相鄰運算節點逐一套用序列數據，實現 Map 操作邏輯
- **RangeNode（序列生成器）**: 透過 `start..end` 語法生成等差數列，可與 ForEachNode 配合使用
- **ButtonNode（觸發節點）**: 手動發送電流觸發訊號的實體按鈕節點
- **GateNode（閘門節點）**: 接收電流時，若輸入值非零則放行訊號通過
- **AppendNode（附加節點）**: 將數據追加寫入相鄰的 TextNode，支援磁性吸附自動配對
- **磁性鄰近邏輯 (Magnetic Proximity Logic)**:
    - 節點拖近彼此時自動建立空間感應連線（Implicit Edges）
    - 支援「吸附觸發」、「吸附執行」等無連線的隱式電流傳遞
- **UI 設計規範文件**: 建立 `UI_UX_GUIDELINES.md`，統一說明節點設計原則、色彩系統與互動規範

### ⚙️ 底層更新 (Infrastructure)
- **隱式邊（Implicit Edges）系統**: 在 Zustand Store 中實作鄰近連線的建立與清除機制
- **CalculateNode 陣列支援**: 當輸入為陣列時，CalculateNode 自動對每個元素執行計算並輸出陣列結果

## [0.3.0] - 2026-03-15


### ✨ 新功能 (Added)
- **核心 TextNode 邏輯化**: 將 TextNode 轉型為主要邏輯與數據紀錄中心。
    - **數據膠囊系統 (Data Pills)**: 支援 `[val](math)` 語法（快捷鍵 `[[val]]`），渲染為美觀且可點擊編輯的數據輸出。
    - **觸發按鈕 (Trigger Buttons)**: 支援 `[label](trigger)` 控制語法，可直接在文本中嵌入手動電流觸發按鈕。
    - **全自動數據包裝**: 當 TextNode 接收外部輸入時，會自動將其包裝為 `[[val]]` 膠囊，保持數據的一致性與可編輯性。
- **智能接環自動化**:
    - **拖放式電流生成**: 從觸發孔拖出的連線，只要丟在目標節點任意位置，即會自動在該處生成 `trigger-in` 孔位。
    - **孔位持久化掃描**: 改進 DOM 掃描算法，在自動同步輸出的同時，完整保留手動添加或預設的輸入孔位。
- **流程優化**:
    - **單點編輯 (Quick-Edit)**: TextNode 點擊一次即可進入編輯模式。
    - **MathLive 深層整合**: 公式編輯器支援自動聚焦與全選舊值，實現「一鍵點擊，直接重打」的流暢體驗。

### 🎨 視覺與 UI (Aesthetics)
- **極簡主義排版**: 
    - **智能結果隱藏**: 當運算型節點（Calculate, Calculus）已連接輸出時，會自動隱藏內部的 RESULT 區域，大幅減少雜訊。
    - **移除浮水印**: 隱藏 ReactFlow 右下角標誌，使介面更加專業精緻。
- **膠囊設計升級**: 數據膠囊採用全新的發光動態外觀，並移除多餘的裝飾符號。

### ⚙️ 底層更新 (Infrastructure)
- **結構優化**: 
    - **節點精簡**: 彻底淘汰 redundant 的 `NumberNode`，由功能更強大的 `TextNode` 取而代之。
    - **精準通訊**: 移除已過時的 `modify` 接環類型，簡化電流觸發與參數修改邏輯。
- **錯誤修復**: 修復了外部公式 (EXT) 在 TextNode 多重輸出環境下無法正確抓取數據的問題。

## [0.2.0] - 2026-03-15

### ✨ 新功能 (Added)
- **函數工具節點 (FunctionNode)**: 
    - **變數自動感應**: 根據 LaTeX 公式（如 $a^2+b$）自動抓取變數並生成對應標籤的輸入接環。
    - **符號精確運算**: 整合 Nerdamer 引擎，支持符號運算，可完美保留根號 ($\sqrt{n}$) 與代數結構。
    - **自動執行外推**: 當節點執行（EXEC）且未連接輸出時，系統會自動生成並連接一個 Data 節點顯示結果。
- **全新接環操作 UX**:
    - **位置自由重定**: 支援 **Command (Mac) / Ctrl (Win) + 拖曳** 接環，可沿著節點邊緣自由滑動。
    - **智能側邊直加**: 在節點左側右鍵直接新增 Input，右側則新增 Output，大幅減少選單點擊。
    - **精緻操作面板**: 以具備中文說明的玻璃質感面板取代舊有輪盤，用於選擇特殊類型的接環。
- **對齊輔助線**: 拖曳接環時自動顯示半透明十字參考線，提升排版精確度。

### 🎨 視覺與 UI (Refinements)
- **節點介面精簡**: 移除 Function 節點的冗餘結果欄位，使布局更專注於公式定義。
- **介面穩定性提升**: 移除接環懸浮時的放大特效，減少在密集節點間操作的視覺干擾。
- **即時連線同步**: 採用 `useUpdateNodeInternals` 技術，確保移動接環時連線（Edges）能無縫跟隨。

### ⚙️ 底層更新 (Infrastructure)
- **Store 邏輯擴充**: 實作 `updateHandle` 行為，支援對接點位置與偏移量的 Partial 更新。
- **結構保護機制**: 導入 `locked` 權限控制，確保受自動管理的接環（如 Function 變數）不會被手動誤刪。

## [0.1.0] - 2026-03-14

### ✨ 新功能 (Added)
- **視覺化節點架構**: 建立核心 React Flow 畫布，支援節點與連線。
- **動態 Handle 系統**: 實作可在節點邊緣任何位置新增/刪除連接點的功能。
- **徑向輪盤選單 (Radial Menu)**: 優化 UX，透過環形介面快速選擇點位類型。
- **自動感應旋轉**: Handle 圖示會根據放置位置自動旋轉（Input 指向內部，Output 指向外部）。
- **數據/工具節點區分**: 
  - `NumberNode`: 支援手動輸入或接收上游數據。
  - `AddNode`: 工具類節點，具備 `EXEC` 手動觸發執行按鈕。
- **右鍵選單增強**: 
  - 空白處右鍵：新增節點。
  - 邊緣右鍵：新增 Handle。
  - 節點右鍵：刪除節點。
- **專案文化**: 導入 **Vibe-coding** 精神與精美 README 文件。

### 🎨 視覺與 UI (Aesthetics)
- **玻璃美學 (Glassmorphism)**: 節點具備半透明毛玻璃質感。
- **動態指標**: 邊緣感應區顯示十字準心 (Crosshair)，提示可互動區域。
- **顏色編碼**: 使用不同邊框樣式區分 數據型 vs 工具型 節點。

### ⚙️ 底層更新 (Infrastructure)
- **GitHub 初始化**: 專案成功推送到 GitHub 遠端倉庫。
- **數學引擎準備**: 安裝 MathLive, Nerdamer, KaTeX 為符號運算版本做準備。
