# 更新日誌 (CHANGELOG)

所有對 **Methmetica** 的重大變更都將記錄在此。

## [0.2.0] - 2026-03-15

### ✨ 新功能 (Added)
- **函數工具節點 (FunctionNode)**: 
    - **變數自動感應**: 根據 LaTeX 公式（如 $a^2+b$）自動抓取變數並生成對應標籤的輸入接環。
    - **符號精確運算**: 整合 Nerdamer 引擎，支持符號運算，可完美保留根號 ($\sqrt{n}$) 與代數結構。
    - **自動執行外推**: 當節點執行（EXEC）且未連接輸出時，系統會自動生成並連接一個 Data 節點顯示結果。
- **全新接環操作 UX**:
    - **位置自由重定**: 支援 **Command (Mac) / Ctrl (Win) + 拖曳** 接環，可沿著節點邊緣自由滑動。
    - **智能側邊直加**: 在節點左側右鍵直接新增 Input，右側則新增 Output，大幅減少選單點擊。
    - **精緻操作面板**: 以具備中文說明的玻璃質感面板取代舊有輪盤，用於選擇特殊類型的接環。
- **對齊輔助線**: 拖曳接環時自動顯示半透明十字參考線，提升排版精確度。

### 🎨 視覺與 UI (Refinements)
- **節點介面精簡**: 移除 Function 節點的冗餘結果欄位，使布局更專注於公式定義。
- **介面穩定性提升**: 移除接環懸浮時的放大特效，減少在密集節點間操作的視覺干擾。
- **即時連線同步**: 採用 `useUpdateNodeInternals` 技術，確保移動接環時連線（Edges）能無縫跟隨。

### ⚙️ 底層更新 (Infrastructure)
- **Store 邏輯擴充**: 實作 `updateHandle` 行為，支援對接點位置與偏移量的 Partial 更新。
- **結構保護機制**: 導入 `locked` 權限控制，確保受自動管理的接環（如 Function 變數）不會被手動誤刪。

## [0.1.0] - 2026-03-14

### ✨ 新功能 (Added)
- **視覺化節點架構**: 建立核心 React Flow 畫布，支援節點與連線。
- **動態 Handle 系統**: 實作可在節點邊緣任何位置新增/刪除連接點的功能。
- **徑向輪盤選單 (Radial Menu)**: 優化 UX，透過環形介面快速選擇點位類型。
- **自動感應旋轉**: Handle 圖示會根據放置位置自動旋轉（Input 指向內部，Output 指向外部）。
- **數據/工具節點區分**: 
  - `NumberNode`: 支援手動輸入或接收上游數據。
  - `AddNode`: 工具類節點，具備 `EXEC` 手動觸發執行按鈕。
- **右鍵選單增強**: 
  - 空白處右鍵：新增節點。
  - 邊緣右鍵：新增 Handle。
  - 節點右鍵：刪除節點。
- **專案文化**: 導入 **Vibe-coding** 精神與精美 README 文件。

### 🎨 視覺與 UI (Aesthetics)
- **玻璃美學 (Glassmorphism)**: 節點具備半透明毛玻璃質感。
- **動態指標**: 邊緣感應區顯示十字準心 (Crosshair)，提示可互動區域。
- **顏色編碼**: 使用不同邊框樣式區分 數據型 vs 工具型 節點。

### ⚙️ 底層更新 (Infrastructure)
- **GitHub 初始化**: 專案成功推送到 GitHub 遠端倉庫。
- **數學引擎準備**: 安裝 MathLive, Nerdamer, KaTeX 為符號運算版本做準備。
